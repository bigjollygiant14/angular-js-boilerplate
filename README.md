#Angular JS Boilerplate

##Stack
* Webpack 2
* Angular JS
  * Angular UI Router

##Install
1. Clone Repo
2. Run `npm install` (or `pnpm install`)
3. Run `npm start`
4. Open browser to localhost:3000

##StyleGuide
Using John Papa's style guide (https://github.com/johnpapa/angular-styleguide/tree/master/a1)
